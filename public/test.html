Step 1: Fix .htaccess

Current .htaccess is blocking PHP files. We need to create a minimal .htaccess:

Create/Replace public/.htaccess with:

RewriteEngine On

# Direct file access for existing files
RewriteCond %{REQUEST_FILENAME} -f
RewriteRule .* - [L]

# Laravel routes for non-existing files
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php [L]

# MIME Types
AddType text/css .css
AddType application/javascript .js
AddType image/png .png
AddType image/jpeg .jpg
AddType image/gif .gif
AddType image/webp .webp
